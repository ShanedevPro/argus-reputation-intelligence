# Sample Reports

This directory contains sanitized example outputs from Argus Reputation Intelligence.

The samples demonstrate report structure, evidence presentation, and HTML/PDF/Markdown output. They are not complete public-opinion datasets and do not include raw provider responses, raw comments, crawl JSON, runtime state, or local logs.

本目录包含 Argus 声誉智能的已脱敏样例报告，用于展示报告结构、证据呈现方式，以及 HTML/PDF/Markdown 输出效果。样例不代表完整舆情覆盖，也不包含 provider 原始响应、原始评论、爬取 JSON、运行时状态或本地日志。

## Included Samples / 已包含样例

- `wang-hedi-artist-profile/`: artist/public-figure reputation research example.
- `xiaomi-su7-enterprise-pr/`: enterprise PR and brand-response research example.

- `wang-hedi-artist-profile/`：明星/公众人物声誉研究样例。
- `xiaomi-su7-enterprise-pr/`：企业公关与品牌回应研究样例。

Each sample contains:

- `report.html`: browser-readable report and the primary visual demo artifact.
- `report.md`: portable Markdown report.
- `report.pdf`: exported PDF for review and sharing.

每个样例包含：

- `report.html`：浏览器可读报告，也是主要视觉演示产物。
- `report.md`：便于阅读和二次处理的 Markdown 报告。
- `report.pdf`：便于审阅和分享的 PDF 导出。

## What To Look For / 重点看什么

- Chinese executive brief with concrete event framing.
- Evidence boundary and sample limitations.
- Key evidence tables instead of raw crawl dumps.
- Scenario-specific recommendations for public-figure and enterprise PR analysis.
- HTML for product inspection, PDF for sharing, Markdown for review.

- 带有具体事件框架的中文高管简报。
- 证据边界与样本限制说明。
- 以关键证据表呈现，而不是原始爬取数据堆叠。
- 面向公众人物和企业公关场景的针对性建议。
- HTML 用于产品检视，PDF 用于分享，Markdown 用于审阅。
