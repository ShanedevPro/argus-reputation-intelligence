# Wang Hedi Artist Profile Sample

This sample demonstrates an artist/public-figure reputation research report generated by Argus.

It is included as a product example for report structure, Chinese narrative quality, evidence tables, and export formats. It is not a claim of complete public-opinion coverage.

Files:

- `report.html`: browser-readable report and primary visual artifact.
- `report.md`: Markdown report.
- `report.pdf`: PDF export.

The sample is sanitized and excludes raw crawl data, raw comments, provider responses, runtime state, local logs, and private paths.

样例已脱敏，不包含原始爬取数据、原始评论、provider 响应、运行时状态、本地日志或私有路径。
