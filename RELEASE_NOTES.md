# Argus Open Source Preview Notes

This zip is a clean preview package for creating a public GitHub repository for Argus Reputation Intelligence.

## Included

- Argus Flask backend and Next.js workbench frontend.
- Restored BettaFish-derived multi-engine analysis chain.
- Sanitized Wang Hedi and Xiaomi SU7 sample reports in HTML, Markdown, and PDF.
- Workbench walkthrough video and screenshots.
- GPLv2 license, NOTICE, SECURITY, CONTRIBUTING, and GitHub templates.

## Not Included

- API keys or `.env` files.
- Raw TikHub/provider crawl responses.
- Runtime databases, logs, caches, local reports, or development artifacts.
- Private planning docs or internal collaboration docs.

## Recommended First GitHub Steps

1. Create a new empty GitHub repository named `argus-reputation-intelligence`.
2. Unzip this package locally.
3. Initialize Git and commit the contents as `Initial open source release`.
4. Review README, sample reports, license, and attribution before making the repo public.
5. Add provider keys only in local `.env` or deployment secret settings.
